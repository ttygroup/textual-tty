# textual-tty

A terminal emulator for Textual apps.

## Demo

```bash
uvx textual-tty
```

## Usage

There's 3 widgets:

1. `textual_tty.Terminal`, a tty that you can use
2. `textual_tty.Program`, launch a program
3. `textual_tty.TerminalProgram`, a terminal emulator in a window
